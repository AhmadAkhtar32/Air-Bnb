# Airbnb Clone - Flutter App

A simple Airbnb-like app for semester project with basic UI screens and navigation.

## Features

- Login screen with validation
- Signup screen
- Forgot password screen
- Main screen with property listings
- Gallery screen
- Wishlist screen
- Profile screen
- Navigation between screens
- Basic form validations

## How to Run

1. Make sure you have Flutter installed on your system
2. Clone this repository
3. Navigate to the project directory
4. Run `flutter pub get` to install dependencies
5. Run `flutter run` to start the app on your connected device/emulator

## Screens

- Login Screen: Initial screen with email/password fields
- Signup Screen: Form to create a new account
- Forgot Password: Screen to request password reset
- Main Screen: Shows property listings with search and categories
- Gallery Screen: Displays properties in a grid view
- Wishlist Screen: Shows saved properties
- Profile Screen: Displays user information and settings

## Dependencies

- flutter: UI framework
- cupertino_icons: For iOS-style icons

## Notes

This is a frontend-only implementation with no backend functionality. All navigation and UI updates are handled within the app.
